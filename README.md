# Cpp Learning

本仓库用于记录我系统学习 C++ 的代码和笔记。

## 学习目标

* 掌握 C++ 基础语法
* 理解函数、数组、指针、引用
* 掌握类和对象、构造函数、析构函数
* 学习继承、多态、模板和 STL
* 为后续数据结构、算法和项目开发打基础

## 目录规划

* chapter01-basic：C++ 基础语法
* chapter02-function：函数
* chapter03-array-pointer-reference：数组、指针和引用
* chapter04-class-object：类和对象
* chapter05-inheritance-polymorphism：继承和多态
* chapter06-template：模板
* chapter07-stl：STL
* notes：学习笔记
