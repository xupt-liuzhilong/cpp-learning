# Date 日期类

本目录用于存放 C++ 日期类的实现代码。

## 当前实现内容

- 构造函数
- 日期打印函数
- 比较运算符重载
  - ==
  - !=
  - >
  - >=
  - <
  - <=
- 日期加减天数
  - Date + int
  - Date - int
  - Date += int
  - Date -= int
- 前置/后置 ++
- 前置/后置 --
- 日期减日期，计算两个日期之间相差天数

## 涉及知识点

- 类和对象
- this 指针
- const 成员函数
- 运算符重载
- 引用返回
- 临时对象
- 前置与后置自增自减区别
