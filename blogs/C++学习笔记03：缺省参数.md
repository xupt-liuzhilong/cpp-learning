# 一、什么是缺省参数

缺省参数也叫默认参数。

他的意思是：再函数声明或定义时，给函数设置一个默认值。

当调用函数时：

- 如果传了实参，就是使用传入的值；
- 如果没有传实参，就使用默认值。

例如：

```cpp
#include <iostream>
using namespace std;

void Func(int a = 0) {
	cout << a << endl;
}

int main() {
	Func();		// 没有传参，使用默认值0
	Func(10);	// 传了参数，使用10

	return 0;
}
```

# 二、全缺省参数

如果函数的所有参数都有默认值，就叫**全缺省参数**。

```cpp
#include <iostream>
using namespace std;

void Func(int a = 1, int b = 2, int c = 3) {
	cout << a << " " << b << " " << c << endl;
}

int main() {
	Func();
	Func(10);
	Func(10, 20);
	Func(10, 20, 30);

	return 0;
}
```

运行结果：

```cpp

10 2 3

```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

可以看到，传了几个参数，前几个参数就使用传入的值，后面没有传的参数使用默认值。

# 三、半缺省参数

如果函数只有一部分参数有默认值，就叫**半缺省参数**。

```cpp
#include <iostream>
using namespace std;

void Func(int a, int b = 2, int c = 3) {
	cout << a << " " << b << " " << c << endl;
}

int main() {
	Func(10);
	Func(10, 20);
	Func(10, 20, 30);

	return 0;
}
```

# 四、缺省参数规则

缺省参数有一个非常重要的规则：

> 缺省参数必须从右往左连续给出。

正确写法：

```cpp
void Func(int a, int b = 10, int c = 20);
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

错误写法：

```cpp
void Func(int a = 10, int b, int c = 20);
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

原因是C++函数调用时，参数是从左往右传递的。

比如：

```cpp
Func(1,2);
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

编译器会认为：

```cpp
a = 1;
b = 2;
c = 使用默认值;
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

如果中间参数没有默认值，就会导致编译器无法判断参数应该怎么匹配。

# 五、调用时不能跳着传参

例如：

```cpp
void Func(int a = 1, int b = 2,int c = 3);
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

可以这样调用：

```cpp
Func();
Func(10);
Func(10, 20);
Func(10, 20, 30);
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

但是不能这样写：

```cpp
Func(, 20);
Func(10, , 30);
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

也就是说：

> 调用参数时，实参必须从左往右依次传递。

# 六、声明和定义不要同时写默认值

如果函数只有声明，没有单独定义，可以直接接写：

```cpp
void Func(int a = 10) {
    cout << a << endl;
}
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

但如果函数声明和定义分开写，一般默认只写在声明中，定义不要再写。

推荐写法：

```cpp
#include <iostream>
using namespace std;

// 函数声明
void Func(int a = 10);

// 函数定义
void Func(int a) {
	cout << a << endl;
}

int main() {
	Func();
	Func(20);

	return 0;
}
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

不要写成：

```cpp
void Func(int a = 10);

void Func(int a = 10) {
	cout << a << endl;
}
```

![](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw== "点击并拖拽以移动")

这样容易造成重复定义默认参数的问题。

# 七、小结

本篇主要学习了C++中的缺省参数。

需要记住：

1. 缺省参数就是给函数设置默认值；
2. 调用函数时，传参就是用传入值，不传参就是用默认值；
3. 所有参数都有默认值，叫全缺省参数；
4. 部分参数有默认值，叫半缺省参数；
5. 缺省参数必须从右往左连续给出；
6. 调用函数时，参数必须从左往右传递，不能跳传；
7. 生明和定义分开写时，默认值一般写在生明中。

缺省参数是C++比C语言更灵活的地方之一，后面学习的类和对象、构造函数时也经常用到。
