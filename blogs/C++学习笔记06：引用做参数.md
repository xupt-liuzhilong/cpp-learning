# C++学习笔记06：引用做参数

# 一、引用简单回顾

引用是C++中给变量起别名的一种语法。

基本写法是：

```cpp
int a = 10;
int& ra = a;
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

这里ra就是a的引用，也可以理解成a的另一个名字。

通过r修改数据，本质上就是修改a。

```cpp
ra = 20;
cout << a << endl;// 20
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

所以引用的最大特点是：

> 引用不是新变量，而是原变量的别名。

# 二、引用做函数参数

引用除了可以给普通变量起别名，还可以作为函数参数使用。

例如：

```cpp
void func(int& x) {
    x = 100;
}
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

这里x是一个引用参数。

当我们调用函数时：

```cpp
int a = 10;
func(a);
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

此时x就相当于是a的别名。

所以函数中修改x，其实就是修改外面的a。

完整代码：

```cpp
#include <iostream>
using namespace std;

void func(int& x) {
    x = 100;
}

int main() {
    int a = 10;

    func(a);

    cout << a << endl;

    return 0;
}
```

# 三、传值调用

在C语言中，我们最常见的是传值调用。

```cpp
#include <iostream>
using namespace std;

void func(int x) {
	x = 100;
}

int main() {
	int a = 10;
	
	func(a);
	
	cout << a << endl;
	
	return 0;
}
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

运行结果：

```cpp
10
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

原因是：

```cpp
void func(int x)
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

这里的x是a的一份拷贝。

函数内部修改的是x，不会影响外面的a。

所以传值调用的特点是：

> 函数内部修改形参，不会影响实参。

# 四、引用形参

引用传参不会床架新的副本，而是让行擦成为实参的别名。

```cpp
void func(int& x) {
    x = 100;
}
```

调用时：

```cpp
int a = 10;
func(a);
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

此时x就是a的别名。

所以修改x，就是修改a。

引用传参的特点是：

> 函数内部可以直接修改外部的变量。

# 五、用引用实现swap函数

交换两个变量是引用传参的经典例子。

如果使用传值写法：

```cpp
#include <iostream>
using namespace std;

void Swap(int x, int y) {
    int tmp = x;
    x = y;
    y = tmp;
}

int main() {
    int a = 10;
    int b = 20;

    Swap(a, b);

    cout << a << " " << b << endl;

    return 0;
}
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

运行结果：

```cpp
10 20
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

可以看到，交换失败了。

因为x和y只是a和b的副本，函数内交换的是副本，不会影响原变量。

------

使用引用传参：

```cpp
#include <iostream>
using namespace std;

void Swap(int& x, int& y) {
    int tmp = x;
    x = y;
    y = tmp;
}

int main() {
    int a = 10;
    int b = 20;

    Swap(a, b);

    cout << a << " " << b << endl;

    return 0;
}
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

运行结果：

```cpp
20 10
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

这次交换成功。

因为x是a的别名，y是b的别名。

在函数交换x和y，本质上就是交换a和b。

# 六、引用传参和指针传参的区别

在C语言中，如果想内部修改外部变量，通常会使用指针。

例如：

```cpp
void Swap(int* p1, int* p2) {
    int tmp = *p1;
    *p1 = *p2;
    *p2 = temp;
}
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

调用时要传地址：

```cpp
Swap(&a, &b);
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

而C++中可以使用引用：

```cpp
void Swap(int& x, int& y) {
    int tmp = x;
    x = y;
    y = tmp;
}
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

调用时直接传变量：

```cpp
Swap(a, b);
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

这两种方式都可以修改实参。

但是引用传参写起来更简单，不需要取得之，也不需要解引用。

简单对比：

| **方式** | **调用方法** | **函数内部** |
| -------- | ------------ | ------------ |
| 指针传参 | Swap(&a, &b) | 需要*p1、*p2 |
| 引用传参 | Swap(a, b)   | 直接使用x、y |

# 七、什么时候适合用引用传参？

引用传参常见的使用场景有两个。

## 1.需要在函数内部修改外部变量

比如交换两个变量：

```cpp
void Swap(int& x, int& y);
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

这种情况使用指针或引用，否则传值无法修改外部变量。

------

## 2.传递较大的对象，避免拷贝

如果参数是一个比较大的结构体或对象，传值产生拷贝，效率较低。

这是可以使用引用传参：

```cpp
void Print(condt Student& s) {
    // 打印学生信息
}
```

![点击并拖拽以移动](data:image/gif;base64,R0lGODlhAQABAPABAP///wAAACH5BAEKAAAALAAAAAABAAEAAAICRAEAOw==)

这里使用const是为了防止函数内部修改s。

这种写法既避免了拷贝，又保证了数据安全。

------

# 八、小结

本篇主要学习了引用做参数。

需要记住：

1. 引用是变量的别名；
2. 引用可以作为函数参数；
3. 传值调用会产生副本，不能修改外部变量；
4. 引用传参不会产生副本，可以直接修改实参；
5. 使用引用可以实现两个变量的交换；
6. 引用传参比指针传参写法更简单；
7. 大对象传参时，常用 const 引用 避免拷贝。

引用做参数是C++中非常常见的写法，后面学习类和对象、拷贝构造、运算符重载时都会经常用到。
